# シャルル の 名刺

My portfolio website

## dependencies

Made in VanillaJS 🍦 with the help of [Vite](https://vitejs.dev/) ⚡

setup
```sh
npm install
```

dev
```sh
npm run dev
```

deploy
```sh
npm run build
```

- Noto Sans Symbol 2 : 400
